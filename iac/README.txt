⚠️ WARNING - AWS DEPLOYMENT SCRIPTS ⚠️

These Infrastructure as Code (IAC) scripts are specifically designed for deploying the application to AWS using CloudFront and S3. 

DO NOT RUN THESE SCRIPTS unless you:
1. Intend to deploy to AWS
2. Understand AWS CloudFront and S3 configuration
3. Have appropriate AWS credentials and permissions
4. Know what you're doing!

Incorrect modifications can lead to:
- Failed deployments
- Broken infrastructure
- Unexpected AWS costs

If you're unsure, consult with your team's cloud infrastructure expert.
